# @black-ink-technologies/bit-workflow-engine

Professional React Workflow Builder (core package) — **theme-only customization** and integrated API layer.

## Installation

### 1. Configure `.npmrc`

To install private packages from **GitHub Packages**, add the following to your project’s `.npmrc` file:

```ini
@black-ink-technologies:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=${GITHUB_AUTH_TOKEN}
```


```bash
npm install @black-ink-technologies/bit-workflow-engine
```

### Setup
```bash
npm install
```

### Build
```bash
npm run build
```

### Lint
```bash
npm run lint
```

### Format
```bash
npm run format
```
